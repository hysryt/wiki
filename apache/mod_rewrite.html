mod_rewrite

URLの置換を行う

RewriteCond

RewriteCond arg1 arg2 [arg3]
arg1 対象文字列（%{変数名}）
arg2 条件パターン（正規表現、オプション）
arg3 フラグ（[OR], [NC], [OR,NC]）

RewriteRule

RewriteRule arg1 arg2 [arg3]
arg1 置換対象（正規表現）
arg2 置換後文字列
arg3 フラグ（[L], [C]など）

基本

RewriteCond 〜     // [1]
RewriteRule 〜     // [1]を満たす時のみ実行される
RewriteCond 〜     // [1]
RewriteRule 〜     // [1]を満たす時のみ実行される
RewriteRule 〜     // 常に実行される
